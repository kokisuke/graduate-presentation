卒業制作テーマ

観光地混雑予測アプリの開発

  • 名古屋市を対象にリアルタイム混雑状況＋予測機能
  • 観光客の分散・混雑緩和・快適な観光を支援

制作の目的

  • 観光地の混雑集中問題（インバウンドの影響）
  • 混雑情報を「見える化」し、観光の質向上を目指す
  • 運営側にも混雑傾向の把握に役立てる

対象地域と展望

  • まずは名古屋市の主要観光地（名古屋城・大須観音など）
  • 全国展開を目指す（京都・大阪・東京・地方都市）

主な機能

  • リアルタイム混雑状況表示
  • 混雑予測機能（曜日・時間・天気・イベントなど）
  • ヒートマップ表示 / 混雑アラート / 空いてる時間の提案

データ収集方法

  • ユーザー投稿型（★1〜5）
  • センサーデータとの連携（将来的）
  • SNS・Webデータ収集（自然言語処理）

使用予定技術

  • HTML / CSS / JavaScript（PWA化）
  • Firebase（リアルタイムDB, 認証）
  • Google Maps API / Chart.js

将来的な展開

  • 自治体や観光庁と連携した地域観光戦略支援
  • 多言語対応でインバウンド支援
  • AIによる混雑予測の高度化
